# 和趣社群店铺驱动的三场景私域变现平台

面向团长与群主规划社群类型建店、审核货盘交易与一级收益闭环，首发交易场景为活动、电商和到店本地服务。电商覆盖物流、自提和虚拟权益交付，首发支持优惠券、固定佣金分享赚与加价帮卖；AI 营销与群接龙后置。本仓库包含产品蓝图、项目会同步材料、历史研究依据与可点击演示原型；真实结算与品类开放仍需通过支付、履约和合规核验。

| `00-原始资料/` | 需求 PDF、业务数据表与现有界面截图 |

## 演示运行

该原型为无构建依赖的静态 HTML 页面，可直接双击仓库根目录的 `index.html` 打开；部署至任意静态站点托管服务时同样以该文件作为入口。
