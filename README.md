# 和趣私域社群变现平台

和趣面向已有社群经营者和零基础新手群主，通过社群冷启动、社群建店、商品库/供给库、开团发布、分销帮卖、订单履约、售后退款和收益结算，帮助用户完成私域社群生意变现。项目进入的主赛道是私域电商与社群团购，延展到本地生活交易；首期用新手冷启动、活动报名、普通团购、卡券团购验证交易闭环，随后再扩展选品中心、团长 App、多平台 C 端小程序、预售团购、到店服务、营销效率和商单能力。

当前权威口径：项目主题是私域社群变现，双通道与多端只作为产品承载方式表达。V1 是同一微信小程序内的 C 端消费通道和 B 端团长/群主经营通道；后续团长 App 是 B 端专业经营工具，微信、小红书、抖音小程序是 C 端消费入口，并共用商品库、开团、订单、售后、归因和收益账本。新手群主是核心客户之一，首发“起号”只定义为社群冷启动，不做公域账号代运营。发布功能正式改为“商品库/供给库 + 开团类型”；选品中心放在 V2 增强版，分销首发只支持一级固定佣金和一级加价帮卖。
