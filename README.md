# 和趣社群团购与本地交易变现平台

和趣面向已有社群经营者和零基础新手群主，规划“社群冷启动 + 社群建店 + 商品库 + 开团发布 + 分销帮卖 + 订单履约 + 售后结算”的社群交易闭环。项目进入的主赛道是私域电商与社群团购，延展到本地生活交易；首期用新手冷启动、活动报名、普通团购、卡券团购验证闭环，随后再扩展选品中心、预售团购、到店服务、营销效率和商单能力。

当前权威口径：新手群主是核心客户之一，首发“起号”只定义为社群冷启动，不做公域账号代运营。发布功能正式改为“商品库/供给库 + 开团类型”，开团类型包括普通团购、预售团购、卡券团购、活动报名和到店服务；选品中心放在 V2 增强版，分销首发只支持一级固定佣金和一级加价帮卖，不做多级分销、保证金、经销商层级和未经核验的自动打款。
