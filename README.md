# 和趣亲子活动平台

面向亲子微信群场景的活动交易与受控一级帮卖平台项目交付仓库。本仓库包含研究规划、项目会决策材料、原始业务资料，以及可点击浏览的产品演示原型。

## 快速查看

- [打开演示原型](./03-演示界面/和趣项目会演示原型.html)
- [阅读项目上下文总览](./README-项目上下文总览.md)
- [查看演示走查脚本](./03-演示界面/演示界面走查脚本.md)
- [阅读持续记忆与协作约束](./docs/PROJECT_MEMORY.md)
- [查看 Airbnb inspired 界面基线](./design-system/README.md)

直接打开仓库根目录的 `index.html` 也会进入演示原型。静态资源已随项目一并提交，不依赖安装或构建步骤。

## 目录

| 目录 | 内容 |
| --- | --- |
| `00-原始资料/` | 需求 PDF、业务数据表与现有界面截图 |
| `01-研究与规划/` | 调研分析、路线图、页面改版规格与产品规划 |
| `02-项目会/` | 决策包、验证计划及会议记录模板 |
| `03-演示界面/` | 单页可交互演示原型、素材与走查脚本 |
| `design-system/` | 所有界面必须复用的 Airbnb inspired 设计 tokens 与说明 |
| `docs/` | 跨会话持续记忆、小范围更新规则与更新日志 |

## 演示运行

该原型为无构建依赖的静态 HTML 页面，可直接双击仓库根目录的 `index.html` 打开；部署至任意静态站点托管服务时同样以该文件作为入口。
