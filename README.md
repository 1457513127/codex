# 和趣私域社群变现平台

和趣面向已有社群经营者和零基础新手群主，通过社群冷启动、社群建店、商品库/供给库、开团发布、分销帮卖、订单履约、售后退款和收益结算，帮助用户完成私域社群生意变现。

项目主赛道是私域社群变现与社群团购，延展到本地生活交易。当前阶段是长期产品策划、项目会材料和静态原型共识阶段，尚未进入真实后端、支付或自动结算实现。

## 如何参与这个项目

本项目适合长期、多专题、多轮讨论，不建议只依赖一个 Codex 对话。推荐方式是：

`一个项目仓库 + 多个专题对话 + 文件化长期记忆`

在 Codex 左侧继续使用当前 `New project`。不要为 PPT、产品、原型分别新建多个项目；应在同一个项目下新建专题对话。建议对话名：

- `00-项目总控与记忆`
- `01-产品定位与商业模式`
- `02-活动变现与货盘`
- `03-商品库与开团发布`
- `04-订单售后与结算`
- `05-分销帮卖与收益`
- `06-PPT与会议材料`
- `07-原型与界面走查`

每个专题对话结束时，把稳定结论写回仓库文件。新对话不依赖旧聊天，只依赖项目文档。

如果你要开启新专题对话，先阅读：

1. [项目协作手册](./00-项目治理/项目协作手册.md)
2. [决策与问题台账](./00-项目治理/决策与问题台账.md)
3. [项目上下文总览](./README-项目上下文总览.md)
4. [项目持续记忆](./docs/PROJECT_MEMORY.md)

## 演示运行

该原型为无构建依赖的静态 HTML 页面，可直接双击仓库根目录的 `index.html` 打开；部署至任意静态站点托管服务时同样以该文件作为入口。
