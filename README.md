# 和趣私域社群变现平台

和趣是面向已有社群经营者和零基础新手群主的私域社群变现平台，目标是把微信群、兴趣圈层和本地社区里的信任关系，转化为可发布、可成交、可履约、可售后、可复购的社群生意。

当前项目仍处于产品策划和共识阶段，尚未进入真实后端、支付分账或自动结算开发。以后新对话、会议和交付都以 **当前产品策划** 为主入口，不再从多份历史蓝图里找口径。

## 当前活跃入口

1. [当前产品策划](./01-研究与规划/当前产品策划.md)：唯一产品口径，包含定位、用户、场景、MVP、多期路线和边界。
2. [项目协作手册](./00-项目治理/项目协作手册.md)：说明 Codex 对话、文件沉淀、交付和小范围更新规则。

## Codex 对话方式

继续使用当前 Codex 左侧的 `New project`，不要为产品、PPT、原型拆多个项目。当前产品策划尚未最终定稿，专题先按策划成熟度划分：

- `00-项目总控与长期记忆`
- `01-产品定位与用户分层`
- `02-市场赛道与竞品研究`
- `03-变现场景与业务主线`
- `04-产品版本与多期路线`
- `05-MVP范围与不做清单`
- `06-功能方案沉淀区`
- `07-PPT与会议材料`
- `08-原型与界面走查`

每个专题对话结束时，只把稳定结论、关键决策或待确认问题写回仓库文件。没有落盘的聊天内容不视为项目事实。
