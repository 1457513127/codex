## 稳定结论

| 项目 | 当前约束 |
| --- | --- |
| 产品定位 | 面向团长/群主的社群店铺驱动私域变现平台：活动、电商、到店本地服务 |
| 当前建设口径 | 选择社群主类型与标签生成专属店铺；从审核货盘发布或选卖；三场景受控真实首发 |
| 调研后的首发优先级 | 活动变现为主验证场景，电商变现为对照验证场景，本地服务为受控补充场景 |
| 首发交易能力 | 活动票务；电商物流/自提/虚拟权益；到店预约核销服务；优惠券；一级固定佣金或加价帮卖 |
| 当前结算口径 | 平台收入为团长订阅 + 按实付额抽佣；优惠承担与退款按比例冲回；真实自动结算前须核验既有链路 |
| 当前不做 | 多级分销、上门/高责任服务、食品生鲜、复杂内容、AI 营销、群接龙、广告与未核验自动打款 |
| 当前交付形态 | 静态 HTML 可点击演示原型及配套业务/决策文档 |
| 代码仓库 | <https://github.com/1457513127/codex> |
| 默认分支 | `master` |
| Notion 更新日志 | <https://www.notion.so/36c9f99b7c3980d2a516fe864dfd780d> |
